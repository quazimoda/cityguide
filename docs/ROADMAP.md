# Roadmap

## Phase 1: MVP informational city guide — done

- Publish core informational pages for Istanbul visitors, students, and expats.
- Use local TypeScript content files for guides, categories, blog posts, and recommendations.
- Preserve a simple static-first deployment model without a database.

## Phase 2: Production readiness and CI — current

- Add a focused GitHub Actions workflow for linting, type checking, source checks, and builds.
- Centralize site URL configuration for metadata, sitemap, and robots output.
- Document setup, scripts, deployment notes, and environment variables.

## Phase 3: Content quality expansion

- Expand guide coverage and improve editorial depth.
- Review content for accuracy, clarity, and consistency.
- Add more practical recommendations for key Istanbul neighborhoods and visitor needs.

## Phase 4: Newsletter/email provider integration

- Connect subscribe and contact flows to a production email or newsletter provider.
- Add clear consent and delivery expectations for users.

## Phase 5: Admin/content management

- Evaluate whether a CMS or lightweight admin workflow is needed.
- Keep content operations simple until editorial scale requires additional tooling.

## Phase 6: Monetization/partners/advertising

- Explore relevant local partnerships and sponsorship placements.
- Keep recommendations useful and transparent for readers.

## Phase 7: Analytics and SEO growth

- Add privacy-conscious analytics when needed.
- Improve technical SEO, content clusters, and search visibility over time.
